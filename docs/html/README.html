Welcome to pypillometry (v1)!¶

[!NOTE] This is pypillometry version 1. The new development version of pypillometry is version 2 which contains breaking changes and many new functions. If you are looking for the new version, please visit the master branch.

Small updates and bug fixes might still be made to version 1, but no new features will be added.

This package implements functions for the analysis of pupillometric data. Features include preprocessing, blink handling, event-related pupil-dilation, plotting and signal modeling.

  • Github-repository: https://github.com/ihrke/pypillometry/tree/v1

  • Homepage/Documentation: https://ihrke.github.io/pypillometry/v1/html/index.html

The project is maintained by Matthias Mittner matthias.mittner@uit.no

To report a bug or recommend a new feature, please use github’s issue tracker.

For more details about how to contribute, please see the contribution-section in the documentation.
